# Inbox

Temporary landing zone for unclassified captures.

Use this for:
- Quick capture
- Voice-to-text snippets
- Raw thoughts needing triage
- Meeting notes not yet processed
- Email or chat snippets needing filing

Do not use this for:
- Stable knowledge objects
- Long-term task tracking
- Private notes that should never be processed

Agent rules:
- Resolve inbox items into a primary home as soon as possible.
- If an item cannot be filed, note the schema gap.
- Do not leave durable context in `inbox/`.
