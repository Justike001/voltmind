# Daily

Private daily operating log.

Use this for:
- What happened today
- Work log
- Blockers
- Private planning
- Temporary thoughts
- Self-reflection
- Tomorrow planning

Do not use this for:
- Durable project context
- Final deliverables
- Shared company status

Contribution rules:
- `daily/` is private by default.
- Raw daily text must not be published.
- The agent may extract redacted candidates into `contribution/candidates/`.
