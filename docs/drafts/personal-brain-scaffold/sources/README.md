# Sources

Raw materials or pointers to raw materials.

Use this for:
- Meeting transcripts
- Email exports
- Teams thread links
- CRM snapshots
- ERP/MES references
- Ticket references
- Web articles
- PDFs
- API responses

Route raw evidence to the narrowest subdirectory:

- `sources/teams/` — Microsoft Teams chats and channel transcripts.
- `sources/meetings/` — meeting transcripts and raw meeting notes.
- `sources/emails/` — exported emails and message threads.
- `sources/calendar/` — calendar events, invitations, and scheduling records.

Rule:
- Distilled pages summarize and cite.
- Source pages preserve raw material, excerpts, or stable pointers.

Do not use source pages as project pages.
