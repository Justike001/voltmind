# Actions

Smallest executable tasks and agent execution contracts.

Use this for:
- Manual actions
- Agent-assisted actions
- Agent-executable actions
- Scheduled agent actions
- Watch agent actions

Execution modes:
- `manual`
- `agent_assisted`
- `agent_executable`
- `scheduled_agent`
- `watch_agent`

Every action should include:
- Owner
- Due date
- Priority
- Related project/workstream
- Source refs
- Automation eligibility
- Runtime
- Trigger
- Risk level
- Confirmation/approval requirements
- Allowed/blocked tools
- Agent contract
- Writeback rules

Do not store full background here. Store pointers to context pages.
