# Commitments

Promises and obligations.

Use this for:
- Promises you made to others
- Promises others made to you
- Team commitments to customers
- Meeting-generated commitments
- Email-generated commitments

Every commitment should include:
- Promisor
- Promisee
- Related project
- Due date
- Status
- Evidence
- Visibility
- Writeback targets

Commitments are strong candidates for Team Brain promotion.
