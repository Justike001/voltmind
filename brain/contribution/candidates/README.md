# Contribution Candidates

Proposed publishable updates from Personal Brain to Team/Company Brain.

Use this for:
- Decision candidates
- Action candidates
- Risk candidates
- Commitment candidates
- Project updates
- Company updates

Every candidate should include:
- Candidate type
- Proposed shared summary
- Original personal source
- Redacted evidence
- Related entities
- Suggested scope
- Suggested owner
- Review status
- Publish target
- Writeback target
