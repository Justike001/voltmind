# Contribution

Promotion layer from Personal Brain to Team/Company Brain.

Use this for:
- Candidates
- Published records
- Rejected records
- Redacted records
- Reviews
- Promotion rules

Required subdirectories:
- `candidates/`
- `published/`
- `rejected/`
- `redacted/`
- `reviews/`

Rules:
- Meeting pages are the main source of contribution candidates.
- Daily pages may produce candidates, but raw daily text must not be published.
- Private pages never participate.
- Every candidate needs review status, suggested scope, evidence, redaction, target, and writeback.
