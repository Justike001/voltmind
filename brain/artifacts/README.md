# Artifacts

Personal deliverables and outputs.

Use this for:
- Email drafts
- Proposal drafts
- Technical plans
- Prompt designs
- Presentation outlines
- Research reports
- Code design explanations
- Memos

Why this exists:
- Not every useful output is a project.
- Not every draft is an idea.
- Artifacts are deliverables that can link back to projects, meetings, sources, or contribution candidates.
