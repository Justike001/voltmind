# Orgs

Internal organization units.

Use this for:
- Teams
- Departments
- Functions
- Committees
- Working groups
- Durable ownership groups

Do not create separate role pages during MVP. Put role fields in `people/` and `orgs/` frontmatter.

Page purpose:
- Mission
- Scope
- Owner/leads
- Responsibilities
- Current priorities
- Open team-level threads
