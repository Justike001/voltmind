# VoltMind Review Gate

Use this skill whenever another workflow proposes page writes. The default Phase 1 contract is human-reviewed.

## Contract

This skill is the approval boundary for mutating Personal Brain pages generated by other workflow skills.

Before writing, show:

- target slug and page type
- whether the file is new or existing
- source evidence
- frontmatter policy fields
- sensitive or private material
- graph links and timeline entries to add

After approval, write additively with `voltmind put` or the matching MCP operation. If rejected, do not write; if partially approved, write only approved pages.

## Output Format

Return approved, rejected, and deferred page slugs. Include any policy reason that blocked a write.

## Anti-Patterns

- Treating silence as approval.
- Overwriting existing user prose.
- Approving external publish or writeback in Phase 1.
